# mydots 🌸

My personal Arch Linux dotfiles — Hyprland + Quickshell.

## Stack

- **WM**: Hyprland
- **Shell**: Quickshell (QML)
- **Terminal**: Kitty
- **Editor**: Neovim (LazyVim)
- **Theme**: Catppuccin

## Install

git clone git@github.com:Miouxxx/mydots.git ~/mydots
cd ~/mydots
stow --target="$HOME" --adopt .
